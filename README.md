# Smart Coastal Fishing Insight AI

Final project for the Building AI course

---

## Summary

This project uses machine learning to predict fish catch potential and environmental risk in coastal regions based on historical fishing, weather, and population data. It helps fishermen make smarter, data-driven decisions while supporting sustainable fishing practices.

---

## Background

Overfishing and inefficient fishing routes are common problems in coastal economies. Many fishermen rely on intuition rather than data, which leads to wasted resources and environmental pressure.

This project addresses:
* inefficient fishing route planning  
* overfishing in sensitive areas  
* lack of data-driven decision tools for fishermen  

This topic is important because global fish populations are declining and sustainable management is becoming critical.

---

## How is it used?

The system is used by fishermen, environmental agencies, and coastal planners.

Typical usage:
1. User inputs location, weather, and historical catch data  
2. AI model estimates fish abundance and fishing success probability  
3. System outputs recommended fishing zones and risk levels  

It can be used before planning daily fishing trips or for long-term environmental monitoring.
